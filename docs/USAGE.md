# DevSync Usage Guide

This document explains how to use DevSync after installation.

---

## Creating a Project

To create a project:

1. Login to the DevSync dashboard
2. Click **Create Project**
3. Enter the project name
4. Save the project

---

## Adding Team Members

To collaborate with others:

1. Open the project
2. Select **Team Members**
3. Add developer email addresses
4. Send invitations

---

## Synchronizing Files

DevSync automatically synchronizes files when changes occur.

Steps:

1. Edit files in your local environment
2. Save the changes
3. DevSync updates the shared project workspace

---

## Task Management

Developers can manage tasks within a project.

Tasks include:

- Bug fixes
- Feature development
- Documentation updates

---

## Notifications

DevSync sends notifications when:

- A task is assigned
- A file is updated
- A project change occurs
